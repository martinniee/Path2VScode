## 需求

设置git bash 作为默认的命令行终端

## 步骤

配置终端的方式（Configure Terminal Settings）

- 图形界面
- 使用命令面板 （建议）





使用命令面板 （建议）

1. 打开命令面板（`Ctrl + Shift + P `) 或  `设置`>`Command Platte...`
2. 输入 `open settings`，选择` Preference: Open User Settings ( JSON )`
3. 在打开的编辑文件中填入内容
